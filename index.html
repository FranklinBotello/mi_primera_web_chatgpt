¿Qué es el Daltonismo?

El daltonismo es una condición visual que afecta la capacidad de una persona para percibir ciertos colores...

Tipos de Daltonismo

Existen varios tipos de daltonismo, siendo los más comunes:

  • Deuteranomalía: Dificultad para percibir el color verde...
  • Protanomalía: Dificultad para percibir el color rojo...
  • Tritanomalía: Dificultad para percibir el color azul...
